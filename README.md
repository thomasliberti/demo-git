# Demo git
## Présentation
Ce **projet** a pour but de mettre en oeuvre `git`.

## Création d'un projet avec Git
Dans votre dossier lancez la commande :
```sh
git init
```

## ajouter tous les fichiers
```sh
git add *
```

## Créer un commit
Lancez la commande :
```sh
git commit -m "First commit
```
